<Note>
Special Agent 需要 Node 22 或更新版本。
</Note>

## 安装

<Tabs>
  <Tab title="npm">
    ```bash
    npm install -g special-agent@latest
    ```
  </Tab>
  <Tab title="pnpm">
    ```bash
    pnpm add -g special-agent@latest
    ```
  </Tab>
</Tabs>

## 新手引导并运行 Gateway 网关

<Steps>
  <Step title="新手引导并安装服务">
    ```bash
    special-agent onboard --install-daemon
    ```
  </Step>
  <Step title="配对 WhatsApp">
    ```bash
    special-agent channels login
    ```
  </Step>
  <Step title="启动 Gateway 网关">
    ```bash
    special-agent gateway --port 18789
    ```
  </Step>
</Steps>

完成新手引导后，Gateway 网关将通过用户服务运行。你也可以使用 `special-agent gateway` 手动启动。

<Info>
之后在 npm 安装和 git 安装之间切换非常简单。安装另一种方式后，运行
`special-agent doctor` 即可更新 Gateway 网关服务入口点。
</Info>

## 从源码安装（开发）

```bash
git clone https://github.com/special-agent/special-agent.git
cd special-agent
pnpm install
pnpm ui:build # 首次运行时会自动安装 UI 依赖
pnpm build
special-agent onboard --install-daemon
```

如果你还没有全局安装，可以在仓库目录中通过 `pnpm special-agent ...` 运行新手引导。

## 多实例快速开始（可选）

```bash
SPECIAL_AGENT_CONFIG_PATH=~/.special-agent/a.json \
SPECIAL_AGENT_STATE_DIR=~/.special-agent-a \
special-agent gateway --port 19001
```

## 发送测试消息

需要一个正在运行的 Gateway 网关。

```bash
special-agent message send --target +15555550123 --message "Hello from Special Agent"
```
